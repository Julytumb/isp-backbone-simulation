## Programm runnen
```
cd ns-3-dev && ./ns3 run network-simulation && cd ..
```

## Run everything

chmod +x run_sim.th
./run_sim.th

## Generate files

python3 parser/genfiles.py 10

## Builden
```
./ns-3-dev/ns3 build
```

## Venv aktivieren
```
source .venv/bin/activate
```

## Parser ausführen
```
python parser/ns3parser.py
```
